# No Damage In Shop

![No Damage In Shop Preview](https://github.com/hlohrenz/REPO-NoDamageInShop/raw/main/preview.gif)

## Overview
This mod prevents players from taking damage while they are in the shop level.

It is meant to stop damage from any source in the shop, including enemies, grenades, friendly fire, and other player interactions.

## Who Needs The Mod
All clients need the mod installed for it to work correctly.
