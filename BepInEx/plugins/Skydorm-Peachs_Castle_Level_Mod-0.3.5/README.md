# 🏰 Peach Castle
*Repo mod that adds Peach Castle as a new level. This Level is a 4x4 Level, meaning it is overall bigger in size than others.*
*It is still early in progress, some gimmicks have been made so far and we almost reached 5,5,5,5,5 Modules for Level 1.*
*There are currently only level 1 modules, 2 and 3 are in work now that some gimmicks are completed.*

## 📒 Note

- Everyone is required to install this mod.
- To report Bugs, please visit the Post of this mod on the official Modding Discord. https://discord.com/channels/1344557689979670578/1407814708454490173
- There are 3 Gimmicks currently. On-Off Blocks, Flip Panels & Shrink Platforms (& Moving/Rotating Platforms).
- If you dislike a Level appearing to much feel free to download https://thunderstore.io/c/repo/p/Skydorm/SkyLevelCooldown_Skydorm/ to prevent levels from appearing again before you played X other levels.
