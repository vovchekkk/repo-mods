## MoreShopItems Updated

The build will try to spawn 2 shelves if possible, but most of the time it will be 1, and in some cases 0. Sorry for the wait and Updates and Feature Additions will hopefully resume Soon!

**MoreShopItems_Updated** relies on [NoItemSpawnLimit](https://thunderstore.io/c/repo/p/HeroHanex/NoItemSpawnLimit/) so that all items bought in the shop spawn in the truck.\
If you encounter any issues with [NoItemSpawnLimit](https://thunderstore.io/c/repo/p/HeroHanex/NoItemSpawnLimit/), please let **HeroHanex** know in the [Discord thread](https://discord.com/channels/1344557689979670578/1381368133926912123) or on the [GitHub page](https://github.com/HeroHanex/REPO-NoItemSpawnLimit).\
Huge thanks to **HeroHanex** for creating and maintaining [NoItemSpawnLimit](https://thunderstore.io/c/repo/p/HeroHanex/NoItemSpawnLimit/) <3
