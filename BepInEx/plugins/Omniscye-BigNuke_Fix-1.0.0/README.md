# BigNuke Fix v1.0.0
Compatibility fix that updates BigNuke to work with the current version of REPO.

## Changelog 1.0.0
- Fixed BigNuke so it works again on the current version of REPO and REPOLib.

## What this mod does

BigNuke by TitanVortex is an older valuable mod that adds a nuke as a valuable, but this fix allows it to work again on the current version of REPOLib and REPO.

If you install this mod through a mod manager, it will automatically download the original BigNuke mod alongside this fix because it is listed as a dependency.
This fix then patches the original mod so it works with the current version of REPOLib and the game.

Original mod:
https://thunderstore.io/c/repo/p/TitanVortex/BigNuke/

## What to expect

- Loads BigNuke on current REPO versions.
- Repairs outdated REPOLib compatibility issues with the mod.
- Restores functionality for the original BigNuke package.

## Important note

- The original mod package, **BigNuke** by **TitanVortex**, may still show as deprecated on Thunderstore.
- I cannot control that status on the original page.
- This fix is made to allow the original mod to work again anyway.
