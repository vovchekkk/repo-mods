# The Deeproot Garden

Deeproot garden is a vanilla-themed level. It currently has 40+ tiles, with more planned soon. Has a rare chance for a (configurable) rain event. Many tiles feature a beautiful outdoor sky, and unique foliage.

(At first it may seem that theres a lack of hiding spots, but you can use the stone base of every single hedge to hide! You can hide in some bushes too! Just crouch next to it.)

Please report any and all bugs to me @beaniebe on discord, or in the REPO modding discord thread!
