# Changelog

## Version 1.0.12

* Fixed a bug that prevented crafting/smelting.
* Adjusted offset of objects in Locked Vault module.
* Made chests easier to open.

## Version 1.0.11

* Prevented some bugged items from appearing near the first extraction.
* Removed a collision box that prevented entities from submerging in lava.
* Fixed doors in Locked Vault to now correctly open when unlocked.
* Adjusted stair colliders in Dungeon module.
* Increased the light casted by cave vines and glow lichen.
* Added a skeleton.

## Version 1.0.10

* Added several 5 new modules.
* Fixed collision for stairs.
* Adjusted valuable spawn locations.
* Added new textures for Missing Redstone locations.

## Version 1.0.9

* Updated to be compatible with the REPO Monster Update.

## Version 1.0.8

* Fixed Items not spawning when using the Crafting Table or the Furnace.

## Version 1.0.7

* Fixed Lecterns, Activator Rails and some other blocks that stopped working correctly.
* Fixed prefabs that will hopefully prevent valuables from having their sizes changed.
* Properly attached the lid of the Chest in Minecart.
* Path to Mineshaft in spawn now always opens starting at level 4.

## Version 1.0.6

* Changed Enemy behaviour so they become more active and better chase players through the different levels of the map.
* Improved collision for some stairs.
